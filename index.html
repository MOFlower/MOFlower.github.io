网络基础

在这里，了解网络世界的通信方式，包括 URL、DNS、HTTP、HTML、计算机网络基础等内容，告诉你怎么浏览器怎么定位资源，浏览器又是怎么找到拥有资源的服务器、浏览器如何与服务器进行对话、浏览器通过什么对页面渲染以及浏览器发出的消息在物理世界是如何实现通信的。

  • URL（Uniform Resource Locator 统一资源定位符）

    学习 web 世界里是如何找到资源的？

    http://localhost:5000/hello?name=lkh

    • URL 格式：

      protocol :// hostname[:port] / path / [;parameters][?query]#fragment
      • protocol ： 协议，在 web 中通常使用 HTTP HTTPS，你可以用 FILE 协议访问本地。
      • hostname: 对应访问的主机名。127.0.0.1 或 localhost ：指向本机，127.0.0.1 为本地回环地址，localhost 在本地的 DNS 解析中默认解析为 127.0.0.1。
      • port：每台主机对应多个端口应用
      • path：访问路径
      • ?query：参数为 url 附带的请求参数

  • DNS（Domain Name System 域名系统）

    • 在 IPv4 中，每台主机的网卡都会分配一个 32 位的 IP 地址，相当于每台主机的唯一名字
    • 但是比如 110.242.68.3 的 IP 地址相对于计算机而言，是容易记住的，但是相对于人而言 www.baidu.com 是更容易记住的.(尝试在浏览器直接访问 110.242.68.3)
    • DNS 的作用就是把相对应的域名地址转化为 IP 地址。

  • HTTP (Hyper Text Transfer Protocol)

    HTTP 即超文本传输协议，是 web 世界中的交流语言之一，定义了两台计算机之间的该如何理解对方的信息的格式和语义，以及需要进行相应操作的时序。想了解其他网络协议可以参考下面的引用。

    wiki 应用层

    HTTP 主要包括 HEADER 和 BODY，学习 HTTP 主要学习 HTTP HEADER。

    • HTTP

      • Header

        • 请求行（request line）| 响应行（response line）

          GET /hello HTTP/1.0

          组成：[method] [url] [protocol]

          其中方法为请求的方式，包括 GET、POST、HEAD 等

          method desciptin
          GET 获取资源
          POST 推送资源，资源一般储存在 Body 中
          HEAD 与 GET 方法相同，但响应不返回 Body
          OPTION 返回支持的方法
          …… …… 
          HTTP/1.0 200 OK

          组成：[method] [code] [description]

          其中状态码为服务器对请求的回应

          code description
          200 OK 表明服务器理解请求，并成功返回
          404 NOT FOUND 表示服务器无法找到相应请求页面
          …… ……

          菜鸟教程 状态码

        • 首部行（request header fileds）

          在 HTTP/1.1 中 HOST 为唯一不可省略字段

          filed description examlple
          Host 访问的服务器主机名 Host: localhost:5000
          Connection 是否保持持续连接 Connection: keep alive
          …… …… ……

          其中 Connection 字段在 HTTP/1.1 中默认为持续连接。在 HTTP/1.0 时代，每次请求创建一个 TCP 连接，响应后关闭连接，这样在一个页面有大量的 XMLHttpRequest（XHR）请求时，会增大服务器开销。

      • Body

        Body 是传输信息的载体，可以通过首部行（header fileds）中设置 Content-Type 来表明返回的 Body 类型。

        multipart/form-data 上传文件格式

  • HTML(Hyper Text Markup Language)
